## Build & Run ##

```sh
$ cd quiniela-kangri
$ ./sbt
> container:start
> browse
```

If `browse` doesn't launch your browser, manually open [http://localhost:56474/](http://localhost:56474/) in your browser.

## Dependencies ##

 * [http://www.scala-sbt.org/](scala-sbt)
 * postgresql
